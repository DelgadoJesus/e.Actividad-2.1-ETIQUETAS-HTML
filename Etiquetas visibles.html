5) Las etiquetas <ul> <ol> y <li>:

Estas etiquetas se utilizan para poder crear listas, pero cada una cumple un tipo de funcion para las estas, <ul> para listas sin ordenar, <ol> para listas ordenadas y <li> son los elementos de la lista.

Ejemplos de esta etiqueta:

  • Amarillo
  • Rojo
  • Azul
  1. Primero
  2. Segundo
  3. Tercero

6) Las etiquetas <table> <tr> y <td>:

Estas etiquetas se utilizan para crear las tablas, el <table> para la tabla, el <tr> para las filas y el <td> para las celdas.

Ejemplo de estas etiquetas:

Juan Pablo
Jesús Daniel
